TBLT_CMPC TBLT_CALLBACK BltFuncIx4SortCmp(TBLT_KH *khPtr,
                                          VOID *arg1ptr,
                                          VOID *arg2ptr,
                                          UCHAR *hvBufferPtr,
                                          ULONG hasRecNo);


 khPtr          I:index file control structure
 arg1ptr        I:string 1
 arg2ptr        I:string 2
 hvBufferPtr    I:set to 0
 hasRecNo       I:set to 0
This routine compares the data at arg1ptr with that at arg2ptr using the sort compare code in khPtr->sortCmpCode.

Use this function when you need to compare foreign key values with key values returned, especially for SORT_NLS, when strcmp() won't perform the job needed.

Return: less than 0 if *arg1ptr < *arg2ptr; 0 if equal; greater than 0 if *arg1ptr > *arg2ptr.
