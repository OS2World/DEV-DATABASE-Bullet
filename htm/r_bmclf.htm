TBLT_RETC TBLT_ENTRY BltMemoCloseFile(TBLT_DH *dhPtr);


 dhPtr          IO:data file control structure
This routine closes a memo file.

When closing a DBF data file the DBT memo file is automatically closed if its TBLT_DH.memoHandle is not 0, so this routine is not normally used.

Return: Non-zero indicates an error, otherwise the memo file is closed.
