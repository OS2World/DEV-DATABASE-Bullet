TBLT_RETC TBLT_ENTRY BltIx4LastKey(TBLT_KH *khPtr,
                                   VOID *keyBufferPtr,
                                   ULONG *recNoPtr);


 khPtr          I:index file control structure
 keyBufferPtr   O:the last key
 recNoPtr       O:recNo attached to the last key
This routine gets the last key and its recNo.

See also: BltIx4GetLast().

Return: Non-zero indicates an error, otherwise the last key/recNo is returned.
