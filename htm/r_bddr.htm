TBLT_RETC TBLT_ENTRY BltDataDebumpRecord(TBLT_DH *dhPtr,
                                         ULONG recNo);


 dhPtr          IO:data file control structure
 recNo          I:number of record to remove
This routine removes the last physical record from the DBF data file.

If a memo field is being used, all memos associated with this record should first be deleted with BltMemoDelete().

Only the last physical record can be removed by this routine (where recNo == TBLT_DH.noRecords). To remove other records use BltDataMarkRecord() or its macro BltDataDeleteRecord(), and follow that with BltDataPack().

Return: Non-zero indicates an error, otherwise all trace of the record at recNo, which is always the last record, is removed.
