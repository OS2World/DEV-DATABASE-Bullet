TBLT_RETC TBLT_ENTRY BltIx4ZapHeader(TBLT_KH *khPtr);


 khPtr          IO:index file control structure
This routine permanently deletes all keys from the IX4 file.

Return: Non-zero indicates an error, otherwise the IX4 file is now empty.
