TBLT_RETC TBLT_ENTRY BltMemoReadHeader(TBLT_DH *dhPtr);


 dhPtr          IO:data file control structure
This routine reads the memo header from disk.

When reading the header of a DBF data file the DBT memo file's header is automatically read so this routine is not normally used.

Return: Non-zero indicates an error, otherwise the memo file's header is read.
