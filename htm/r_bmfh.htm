TBLT_RETC TBLT_ENTRY BltMemoFlushHeader(TBLT_DH *dhPtr);


 dhPtr          IO:data file control structure
This routine flushes a memo file.

When flushing a DBF data file the DBT memo file is automatically flushed so this routine is not normally used.

Return: Non-zero indicates an error, otherwise the memo file is flushed.
