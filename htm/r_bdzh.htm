TBLT_RETC TBLT_ENTRY BltDataZapHeader(TBLT_DH *dhPtr);


 dhPtr          IO:data file control structure
This routine permanently deletes all data records from the DBF file.

If a memo file is attached it is also zapped. removing all data from it.

Return: Non-zero indicates an error, otherwise the DBF file is now empty.
