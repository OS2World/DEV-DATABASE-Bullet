TBLT_RETC TBLT_ENTRY BltIx4GetFirst(TBLT_KH *khPtr,
                                    VOID *keyBufferPtr,
                                    ULONG *recNoPtr,
                                    VOID *recordPtr);


 khPtr          I:index file control structure
 keyBufferPtr   O:the first key
 recNoPtr       O:recNo of record data of first key
 recordPtr      O:record data of first key
This routine gets the first key and its recNo and its data record.

If the TBLT_DH.flags bit FLAGS_SKIP_DELETED_BIT, of khPtr->xbLinkPtr->flags, is set, this routine skips over all data records marked with a * (star) in the delete tag field (the first byte of the data record).

Return: Non-zero indicates an error, otherwise the first key/recNo/recordPtr is returned.
